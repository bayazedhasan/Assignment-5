Question-1: What are the different ways to select an element in the DOM?

Ans: There are 5 ways in which you can select elements in a DOM using selectors.
1. getElementsByTagName() 2. getElementsByClassName() 3. getElementById() 4. querySelector() 5. querySelectorAll()

Question-2: What is the difference between innerHTML, innerText, and textContent ?

Ans: TextContents is all text contained by an element and all its children that are for formatting purposes only.
InnerText returns all text contained by an element and all its child elements.
InnerHtml returns all text, including html tags, that is contained by an element.

Question-3: What is event delegation in the DOM?

Ans: Event delegation is a technique where you attach a single event listener to a parent element, allowing it to handle events triggered on its child elements, instead of adding separate listeners to each child element.

Question-4: What is event bubbling in the DOM?

Ans: Event bubbling refers to the process where an event triggered on a child element propagates upwards through the DOM hierarchy, triggering event handlers on its parent elements until it reaches the topmost element.

Question-5: How do you create, add, and remove elements using JavaScript?

Ans: To add elements at a specific index, you can use the splice method. The splice method allows you to add and remove elements from a specific index in an array.
